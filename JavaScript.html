Associative Arrays

An associative array in JavaScript is one with numbered indexes. It will use 1 variable name, but different numbers to distinguish between different values. For example, the associate array name can be var animal, and then the list will go by numbers in brackets.
So we can have an associate array list look like this:
animal[0] = "pigeon"
animal[1] = "Turkey"
animal[2] = "Chicken"

JavaScript Objects

A JavaScript object is like a giant variable that is able to contain multiple properties with different values.

One example of this is a home. If I were to describe to someone the different aspects of different homes, I would list out different properties.

First, we'll need to know what type of home it is (if it's an apartment or a house).
Then we may want to know how many bedrooms are in the home.
If we want to travel to this home, we will need to know its address.
And if we wish to purchase this home, we need to know how much it costs.
